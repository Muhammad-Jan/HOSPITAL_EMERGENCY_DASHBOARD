# üè• Hospital Emergency Analysis Dashboard

I developed a **Hospital Emergency Analysis Dashboard** in Excel to analyze patient flow, emergency severity, wait times, admissions, demographics, and ER operational performance.  
This dashboard uses **Power Query, advanced Excel formulas, Pivot Tables, and interactive visualizations** to provide actionable insights for hospital administrators and emergency departments.

The project includes **data preprocessing, cleaning, modeling, and visualization** to optimize emergency room operations and enhance patient experience.
